PicShow

About

PicShow is a simple tool to overlay an image onto any part of the screen.

Instructions

Extract PicShow.zip into a folder. Double-click on PicShow.jar to launch it with Java. You can drag the window anywhere. Right-click on the window that pops up to access the menu. WARNING: THIS PROGRAM WILL NOT WORK WITH JAVA 8. INSTALL THE LATEST VERSION HERE.
